# Crypto CTF 기타

여기서는 crypto challenges에서 자주 등장하지만 다른 곳에 깔끔하게 들어맞지 않는 잡다한 페이지들을 모아둡니다.

## 난해한 언어

### 기법

crypto 작업이 사실상 esolang 프로그램을 실행하고 그 출력을 디코딩하는 것일 때 이 기법을 사용하세요.

문제가 표준 언어처럼 보이지 않는 코드를 제공한다면:

- esolang를 식별하세요 (특징적인 토큰을 Google 검색).
- 온라인 interpreter를 사용하거나 Docker image를 사용하세요.
- 출력이 이상하면 실행 후 다중 레이어의 인코딩/압축이 있는지 확인하세요.

유용한 시작점:
