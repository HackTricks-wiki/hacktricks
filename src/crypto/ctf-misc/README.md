## Langages ésotériques

### Technique

Utilisez ceci lorsqu'une tâche crypto consiste en réalité à : exécuter un programme esolang puis décoder sa sortie.

Si un challenge vous donne du code qui ne ressemble pas à un langage standard :

- Identifiez l'esolang (Googlez un token distinctif).
- Utilisez un interpréteur en ligne ou une image Docker.
- Si la sortie est étrange, cherchez un encodage/compression en couches après l'exécution.
