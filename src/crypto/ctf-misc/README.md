## Linguagens esotéricas

### Técnica

Use isto quando uma tarefa de crypto for, na verdade, executar um programa esolang e depois decodificar o que ele produz.

Se um desafio lhe der code que não parece uma linguagem padrão:

- Identifique o esolang (Google por um token distintivo).
- Use um interpretador online ou uma imagem Docker.
- Se a saída for estranha, procure por camadas de codificação/compressão após a execução.
