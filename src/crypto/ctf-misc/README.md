## 奇异语言

### 方法

当一个 crypto 题目实际上是：运行一个 esolang 程序然后解码其输出时，使用此方法。

如果某个 challenge 给你的 code 看起来不像标准语言：

- Identify the esolang (Google a distinctive token).
- Use an online interpreter or a Docker image.
- 如果输出很奇怪，执行后检查是否存在分层的 encoding/compression。
