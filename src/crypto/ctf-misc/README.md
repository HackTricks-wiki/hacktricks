# Crypto CTF Misc

Колекція сторінок, які часто зустрічаються у crypto challenges, але не вписуються в інші розділи.

## Езотеричні мови

### Техніка

Використовуйте це, коли crypto-завдання насправді: запустити esolang-програму і потім декодувати те, що вона виводить.

Якщо завдання дає вам code, який не схожий на стандартну мову:

- Визначте esolang (шукайте в Google відмітний токен).
- Використовуйте online interpreter або Docker image.
- Якщо output дивний, шукайте багатошарову encoding/compression після виконання.
