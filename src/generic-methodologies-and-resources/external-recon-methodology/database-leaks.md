## 데이터 유출 검색 엔진

- [greynoise](https://viz.greynoise.io/) - IPs, Tags, CVEs, vpn, dns 등을 검색
- [Dehashed](https://www.dehashed.com/) - 이메일이 어떤 데이터베이스에서 leak되었는지 검색할 수 있습니다.
- [HaveIbeenPwned?](https://haveibeenpwned.com/) - 이메일 주소가 데이터 유출에 포함되었는지 확인
- [ScamSearch](https://scamsearch.io/) - 전화번호, 이메일, 프로필이 사기인지 검색
- [Intelligence X](https://intelx.io/) - Intelligence X는 검색 엔진이자 데이터 아카이브입니다. · 검색은 selectors(선택자)를 사용하며 이메일 주소, 도메인, URL, IPs 등과 같은 특정 검색어로 작동합니다.
- [spycloud](https://portal.spycloud.com/endpoint/enriched-stats/[YOUR-MAIL]) - YOUR-MAIL 자리에 이메일을 입력하세요.
- [weleakinfo](https://weleakinfo.io/) - We Leak Info - Leaked Dehashed Databases, leaks를 검색
- [breachdirectory](https://breachdirectory.org/) - 이메일 또는 사용자 이름이 노출되었는지 확인
- [leakcheck](https://leakcheck.io/) - 자격증명(credentials)이 침해되었는지 확인
- [Findemail.io](https://findemail.io/) - 특정 회사의 이메일 주소 찾기
- [LibraryOfLeaks](https://search.libraryofleaks.org) - leak 문서, 회사, 사람, 데이터베이스 등을 검색
- [LeakRadar](https://leakradar.io/) - 2 B+ 평문 info-stealer credentials에 대한 즉시 검색; 이메일, 도메인, 메타데이터 쿼리, 모니터링 및 API
- [InfoStealers](https://infostealers.info/en/info) - 다크넷에 노출된 infostealer 로그를 인덱싱하여 보안팀, 조사관, 연구자 및 디지털 포렌식 전문가들이 검색하고 활용할 수 있게 합니다.
- [Leak-Lookup](https://leak-lookup.com/) - 3000+ 데이터베이스에서 30억+ 레코드를 보유한 데이터 유출 검색 엔진
- [BreachDirectory](https://breachdirectory.org/) - 자격증명(credentials)이 데이터 유출에서 침해되었는지 확인
- [Scylla.so](https://scylla.so/) - 커뮤니티 기반 데이터 유출 검색 엔진
- [Leaked.domains](https://leaked.domains/) - 도메인 관련 leaked credentials 및 데이터 검색
- [WhiteIntel](https://whiteintel.io/) - 위협 인텔리전스를 위한 다크웹 데이터 leak 검색 엔진
- [PSBDMP](https://psbdmp.ws/) - Pastebin 덤프 검색 및 모니터링 플랫폼

## 데이터 leaks 열거용 도구

- [Leaker](https://github.com/vflame6/leaker) - 10개 breach 데이터베이스를 동시에 검색하는 Passive leak 열거 CLI 도구.
