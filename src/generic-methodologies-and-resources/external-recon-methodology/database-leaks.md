## Data Breach खोज इंजन

- [greynoise](https://viz.greynoise.io/) - IPs, Tags, CVEs, vpn, dns... के लिए खोजें
- [Dehashed](https://www.dehashed.com/) - आप अपना email खोज सकते हैं यदि यह किसी database में leak हुआ हो।
- [HaveIbeenPwned?](https://haveibeenpwned.com/) - जाँचें कि आपका email address किसी data breach में है या नहीं
- [ScamSearch](https://scamsearch.io/) - खोजें कि कोई phone, email, profile scammer है या नहीं।
- [Intelligence X](https://intelx.io/) - Intelligence X एक search engine और data archive है। · खोज selectors के साथ काम करती है, यानी specific search terms जैसे कि email addresses, domains, URLs, IPs...
- [spycloud](https://portal.spycloud.com/endpoint/enriched-stats/[YOUR-MAIL]) - अपने मेल को YOUR-MAIL में डालें।
- [weleakinfo](https://weleakinfo.io/) - We Leak Info - Leaked Dehashed Databases, leaks के लिए खोज करें।
- [breachdirectory](https://breachdirectory.org/) - जाँचें कि आपका EMAIL या USERNAME compromised हुआ था या नहीं
- [leakcheck](https://leakcheck.io/) - पता करें कि आपकी credentials compromised हुई हैं या नहीं
- [Findemail.io](https://findemail.io/) - दिए गए company के लिए email addresses खोजें।
- [LibraryOfLeaks](https://search.libraryofleaks.org) - leak documents, companies, people, database और अधिक खोजें
- [LeakRadar](https://leakradar.io/) - 2 B+ plain-text info-stealer credentials पर त्वरित खोज; email, domain, metadata queries, monitoring & API
- [InfoStealers](https://infostealers.info/en/info) - darknet-exposed infostealer logs को index करता है और security teams, investigators, researchers, और digital forensics professionals के लिए searchable और actionable बनाता है।
- [Leak-Lookup](https://leak-lookup.com/) - 3000+ databases से 3 billion+ रिकॉर्ड्स के साथ data breach search engine.
- [BreachDirectory](https://breachdirectory.org/) - जाँचें कि आपकी credentials data breaches में compromised हुई हैं या नहीं।
- [Scylla.so](https://scylla.so/) - Community-driven database breach खोज इंजन।
- [Leaked.domains](https://leaked.domains/) - domain-संबंधित leaked credentials और data खोजें।
- [WhiteIntel](https://whiteintel.io/) - threat intelligence के लिए Dark web data leak खोज इंजन।
- [PSBDMP](https://psbdmp.ws/) - Pastebin dump खोज और मॉनिटरिंग प्लेटफ़ॉर्म।

## Tools to enumerate data leaks

- [Leaker](https://github.com/vflame6/leaker) - Passive leak enumeration CLI tool जो एक साथ 10 breach databases में खोज करता है।
