## 数据泄露搜索引擎

- [greynoise](https://viz.greynoise.io/) - 搜索 IPs、Tags、CVEs、vpn、dns 等...
- [Dehashed](https://www.dehashed.com/) - 可以搜索你的邮箱，查看是否在某些数据库中被 leak。
- [HaveIbeenPwned?](https://haveibeenpwned.com/) - 检查你的邮箱地址是否出现在数据泄露中
- [ScamSearch](https://scamsearch.io/) - 搜索电话号码、邮箱、资料以判断是否为骗子。
- [Intelligence X](https://intelx.io/) - Intelligence X 是一个搜索引擎和数据档案库。· 搜索支持 selectors，即使用特定搜索词，如邮箱地址、域名、URLs、IPs...
- [spycloud](https://portal.spycloud.com/endpoint/enriched-stats/[YOUR-MAIL]) - 在链接中的 YOUR-MAIL 位置填入你的邮箱。
- [weleakinfo](https://weleakinfo.io/) - We Leak Info - Leaked Dehashed Databases，搜索 leaks。
- [breachdirectory](https://breachdirectory.org/) - 检查你的邮箱或用户名是否被泄露
- [leakcheck](https://leakcheck.io/) - 检查你的凭证是否被泄露
- [Findemail.io](https://findemail.io/) - 查找指定公司的邮箱地址。
- [LibraryOfLeaks](https://search.libraryofleaks.org) - 搜索 leak 文档、公司、人员、数据库等信息
- [LeakRadar](https://leakradar.io/) - 对 2 B+ 明文 info-stealer 凭据进行即时搜索；支持 email、domain、metadata 查询、监控与 API
- [InfoStealers](https://infostealers.info/en/info) - 索引 darknet 暴露的 infostealer 日志，使其可被安全团队、调查员、研究人员和数字取证专业人员检索并采取行动。
- [Leak-Lookup](https://leak-lookup.com/) - 一个数据泄露搜索引擎，包含来自 3000+ 数据库的 30 亿+ 记录。
- [BreachDirectory](https://breachdirectory.org/) - 检查你的凭证是否在数据泄露中被窃取。
- [Scylla.so](https://scylla.so/) - 社区驱动的数据库泄露搜索引擎。
- [Leaked.domains](https://leaked.domains/) - 搜索与域相关的 leaked 凭据和数据。
- [WhiteIntel](https://whiteintel.io/) - 用于威胁情报的暗网数据 leak 搜索引擎。
- [PSBDMP](https://psbdmp.ws/) - Pastebin dump 搜索与监控平台。

## 枚举 data leaks 的工具

- [Leaker](https://github.com/vflame6/leaker) - 被动 leak 枚举 CLI 工具，可同时搜索 10 个泄露数据库。
