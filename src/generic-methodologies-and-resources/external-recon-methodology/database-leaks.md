## Moteurs de recherche de data breach

- [greynoise](https://viz.greynoise.io/) - Rechercher des IPs, Tags, CVEs, vpn, dns...
- [Dehashed](https://www.dehashed.com/) - Vous pouvez rechercher votre adresse e-mail si elle est leak dans certaines bases de données.
- [HaveIbeenPwned?](https://haveibeenpwned.com/) - Vérifier si votre adresse e-mail figure dans une violation de données.
- [ScamSearch](https://scamsearch.io/) - Rechercher un numéro de téléphone, e-mail ou profil pour savoir s'il s'agit d'un arnaqueur.
- [Intelligence X](https://intelx.io/) - Intelligence X est un moteur de recherche et une archive de données. · La recherche fonctionne avec des sélecteurs, c.-à-d. des termes de recherche spécifiques tels que adresses e-mail, domaines, URLs, IPs...
- [spycloud](https://portal.spycloud.com/endpoint/enriched-stats/[YOUR-MAIL]) - mettez votre mail dans YOUR-MAIL.
- [weleakinfo](https://weleakinfo.io/) - We Leak Info - Leaked Dehashed Databases, search for leaks.
- [breachdirectory](https://breachdirectory.org/) - VÉRIFIEZ SI VOTRE E-MAIL OU NOM D'UTILISATEUR A ÉTÉ COMPROMIS
- [leakcheck](https://leakcheck.io/) - Découvrez si vos identifiants ont été compromis
- [Findemail.io](https://findemail.io/) - Trouver des adresses e-mail pour une entreprise donnée.
- [LibraryOfLeaks](https://search.libraryofleaks.org) - Rechercher des documents de leak, entreprises, personnes, bases de données et plus
- [LeakRadar](https://leakradar.io/) - Recherche instantanée à travers plus de 2 milliards de credentials plain-text d'info-stealers ; e-mail, domaine, requêtes de métadonnées, monitoring & API
- [InfoStealers](https://infostealers.info/en/info) - Indexe les logs d'infostealers exposés sur le darknet et les rend consultables et exploitables pour les équipes de sécurité, enquêteurs, chercheurs et spécialistes de la criminalistique numérique.
- [Leak-Lookup](https://leak-lookup.com/) - Moteur de recherche de data breach avec plus de 3 milliards d'enregistrements provenant de 3000+ bases de données.
- [BreachDirectory](https://breachdirectory.org/) - Vérifiez si vos identifiants ont été compromis dans des data breaches.
- [Scylla.so](https://scylla.so/) - Moteur de recherche de data breach communautaire.
- [Leaked.domains](https://leaked.domains/) - Rechercher des credentials et des données leaked liées à un domaine.
- [WhiteIntel](https://whiteintel.io/) - Moteur de recherche de data leaks du dark web pour le renseignement sur les menaces.
- [PSBDMP](https://psbdmp.ws/) - Plateforme de recherche et de monitoring de dumps Pastebin.

## Outils pour énumérer les data leaks

- [Leaker](https://github.com/vflame6/leaker) - Outil CLI d'énumération passive de leak qui recherche simultanément dans 10 bases de données de breach.
