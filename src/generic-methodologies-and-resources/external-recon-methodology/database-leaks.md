## Motori di ricerca per Data Breach

- [greynoise](https://viz.greynoise.io/) - Cerca IP, Tag, CVEs, vpn, dns...
- [Dehashed](https://www.dehashed.com/) - Puoi cercare la tua email se è presente in leak di qualche database.
- [HaveIbeenPwned?](https://haveibeenpwned.com/) - Controlla se il tuo indirizzo email è in un data breach
- [ScamSearch](https://scamsearch.io/) - Cerca telefono, email, profilo per verificare se si tratta di uno scammer.
- [Intelligence X](https://intelx.io/) - Intelligence X è un motore di ricerca e un archivio di dati. · La ricerca funziona con selettori, cioè termini specifici come indirizzi email, domini, URL, IP...
- [spycloud](https://portal.spycloud.com/endpoint/enriched-stats/[YOUR-MAIL]) - Inserisci la tua mail in YOUR-MAIL.
- [weleakinfo](https://weleakinfo.io/) - We Leak Info - Leaked Dehashed Databases, cerca leak.
- [breachdirectory](https://breachdirectory.org/) - Controlla se la tua email o username è stata compromessa
- [leakcheck](https://leakcheck.io/) - Scopri se le tue credenziali sono state compromesse
- [Findemail.io](https://findemail.io/) - Trova indirizzi email di una determinata azienda.
- [LibraryOfLeaks](https://search.libraryofleaks.org) - Cerca documenti di leak, aziende, persone, database e altro
- [LeakRadar](https://leakradar.io/) - Ricerca istantanea su 2 B+ di credenziali in plain-text da info-stealer; email, domain, metadata queries, monitoring & API
- [InfoStealers](https://infostealers.info/en/info) - Indicizza i log di infostealer esposti nel darknet e li rende ricercabili e utilizzabili per team di sicurezza, investigatori, ricercatori e professionisti della digital forensics.
- [Leak-Lookup](https://leak-lookup.com/) - Motore di ricerca per data breach con oltre 3 billion record da 3000+ database.
- [BreachDirectory](https://breachdirectory.org/) - Controlla se le tue credenziali sono state compromesse in data breach.
- [Scylla.so](https://scylla.so/) - Motore di ricerca per data breach guidato dalla community.
- [Leaked.domains](https://leaked.domains/) - Cerca credenziali e dati leaked correlati a domini.
- [WhiteIntel](https://whiteintel.io/) - Motore di ricerca per leak di dati nel dark web per threat intelligence.
- [PSBDMP](https://psbdmp.ws/) - Piattaforma per la ricerca e il monitoraggio di dump di Pastebin.

## Strumenti per enumerare data leak

- [Leaker](https://github.com/vflame6/leaker) - Tool CLI di enumerazione passiva dei leak che effettua ricerche contemporanee su 10 breach database.
