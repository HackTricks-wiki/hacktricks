## データ侵害検索エンジン

- [greynoise](https://viz.greynoise.io/) - IPs、Tags、CVEs、vpn、dnsなどを検索
- [Dehashed](https://www.dehashed.com/) - メールが何らかのデータベースで leak しているか検索できます
- [HaveIbeenPwned?](https://haveibeenpwned.com/) - メールアドレスがデータ侵害に含まれているか確認する
- [ScamSearch](https://scamsearch.io/) - 電話番号、メール、プロフィールを検索して詐欺師かどうかを調べる
- [Intelligence X](https://intelx.io/) - Intelligence X は検索エンジン兼データアーカイブです。検索はセレクタ（メールアドレス、ドメイン、URLs、IPsなどの特定の検索語）で動作します
- [spycloud](https://portal.spycloud.com/endpoint/enriched-stats/[YOUR-MAIL]) - YOUR-MAIL にメールアドレスを入れてください
- [weleakinfo](https://weleakinfo.io/) - We Leak Info - Leaked Dehashed Databases、leak を検索
- [breachdirectory](https://breachdirectory.org/) - メールまたはユーザー名が漏洩しているか確認する
- [leakcheck](https://leakcheck.io/) - 資格情報が漏洩しているかを確認する
- [Findemail.io](https://findemail.io/) - 指定した会社のメールアドレスを検索
- [LibraryOfLeaks](https://search.libraryofleaks.org) - leak ドキュメント、企業、人、データベースなどを検索
- [LeakRadar](https://leakradar.io/) - 20億以上のプレーンテキストのinfo-stealer資格情報を瞬時に検索：email、domain、metadataクエリ、監視 & API
- [InfoStealers](https://infostealers.info/en/info) - ダークネットに露出したinfostealerログをインデックス化し、セキュリティチーム、捜査官、研究者、デジタルフォレンジクスの専門家が検索・活用できるようにする
- [Leak-Lookup](https://leak-lookup.com/) - 3000以上のデータベースから30億以上のレコードを持つデータ侵害検索エンジン
- [BreachDirectory](https://breachdirectory.org/) - データ侵害で資格情報が漏洩しているか確認する
- [Scylla.so](https://scylla.so/) - コミュニティ主導のデータベース侵害検索エンジン
- [Leaked.domains](https://leaked.domains/) - ドメイン関連の leaked 資格情報とデータを検索
- [WhiteIntel](https://whiteintel.io/) - 脅威インテリジェンス向けのダークウェブの data leak 検索エンジン
- [PSBDMP](https://psbdmp.ws/) - Pastebinダンプの検索および監視プラットフォーム

## data leaks を列挙するツール

- [Leaker](https://github.com/vflame6/leaker) - 10のブリーチデータベースを同時に検索するパッシブな leak 列挙 CLI ツール
