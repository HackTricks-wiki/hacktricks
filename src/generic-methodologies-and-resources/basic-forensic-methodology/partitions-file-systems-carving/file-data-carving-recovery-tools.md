### Extundelete / Ext4magic (EXT 3/4 undelete)

рдпрджрд┐ рд╕реНрд░реЛрдд рдлрд╝рд╛рдЗрд▓ рдкреНрд░рдгрд╛рд▓реА Linux EXT-рдЖрдзрд╛рд░рд┐рдд рд╣реИ, рддреЛ рдЖрдк рд╣рд╛рд▓ рд╣реА рдореЗрдВ рд╣рдЯрд╛рдП рдЧрдП рдлрд╝рд╛рдЗрд▓реЛрдВ рдХреЛ **рдкреВрд░реНрдг рдХрд╛рд░реНрд╡рд┐рдВрдЧ рдХреЗ рдмрд┐рдирд╛** рдкреБрдирд░реНрдкреНрд░рд╛рдкреНрдд рдХрд░рдиреЗ рдореЗрдВ рд╕рдХреНрд╖рдо рд╣реЛ рд╕рдХрддреЗ рд╣реИрдВред рджреЛрдиреЛрдВ рдЙрдкрдХрд░рдг рд╕реАрдзреЗ рдПрдХ рдкрдврд╝рдиреЗ-рдХреЗ-рд▓рд┐рдП рдЫрд╡рд┐ рдкрд░ рдХрд╛рдо рдХрд░рддреЗ рд╣реИрдВ:
